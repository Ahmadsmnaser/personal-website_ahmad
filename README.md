# Ahmad Naser — Personal Website 🚀

A personal portfolio website built to present my work as a Software Engineer with a strong focus on **low-level systems**, **backend development**, and **AI-augmented engineering workflows**.

This website serves as a central place to showcase my background, selected projects, technical skills, and contact information in a clean and modern interface.

---

## Live Website 🌐



```bash
https://personal-websiteahmad.vercel.app/
```

---

## Overview

The portfolio is designed to communicate three things clearly:

- who I am as an engineer
- what kinds of systems and projects I build
- how to reach me for opportunities

The site highlights my transition from academic and systems-level work into real-world software engineering through projects, internship experience, and practical development work.

---

## 🏗️ Architecture

The application is built using a modern React-based architecture:

- Next.js handles routing and server-side rendering
- Components are structured using a modular design
- Styling is managed through Tailwind utility classes
- Pages are statically generated where possible for performance

The goal is to keep the system lightweight, fast, and maintainable.
